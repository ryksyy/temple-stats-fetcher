## How to run

> 1. Install Node and NPM
> 2. Clone the project
> 3. In `package.json` replace `exampleRsn.txt` with the path+txt file you need to look up or run script with node `index.js [pathToFile]` or replace the contents of `exampleRsn.txt`
> 4. Give it time to run, it will take a minute or three because we dont want to hit Temple rate limit
